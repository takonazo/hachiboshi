Project File

【厳秘】Project:Meteor

八星製薬株式会社 / PJ:Meteorタスクフォース

1. 目的

本計画は、国境危機管理局 第7鑑定室の人員を一部排除し、当社の"輸送活動"に対する監視・解析能力を低下させることを目的とする。

2. 概要

第7鑑定室は、暗号解析および特殊貨物鑑定に優れ、当社の"輸送活動"における主要な障害となっている。 本計画では、新たに確認された物質(X-██)の特性を利用し、鑑定士が貨物を調査する過程で異常を発生させる。 X-██は証拠が残りにくく、既存検査でも特定されにくいため、事案発生後も貨物との因果関係を明確にされにくい。

3. 手段

匿名のタレコミを意図的に流し、特定貨物を国境危機管理局の検査対象へ誘導する。

架空名義による偽装も検討されたが、第7鑑定室に背後関係を解析されれば、かえって疑念を強めるおそれがある。 そのため、貨物は八星製薬名義の通常配送品として扱い、「不審ではあるが、検査しても決定的証拠が出ない」状態を作る。

貨物には、組織員の偶発的被害に備え、緊急対処用の解毒材料および手順書を同封する。 ただし、手順は特定の暗号規則に基づき秘匿する。

4. 実施後対応

国境危機管理局より照会があった場合、当組織は貨物との関連を否定する。 回答は、研究用試料の誤認、外部からの悪質な通報、または調査中の事案として統一する。

また、すべての関連資料は、関係部署の判断で速やか且つ完全に抹消できる状態を保持する。
